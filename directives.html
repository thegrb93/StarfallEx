Preprocessor directives

List of preprocessor directives that are put on SF script's top.

List

--@client Set the processor to only run on the client.
--@include path Mark a file to be included in the upload.
--@includedir path Mark a directory to be included in the upload.
--@model model Set the model of the processor entity.
--@name name Set the name of the script.
--@server Set the processor to only run on the server.
--@client

Set the processor to only run on the client. Shared is default

Usage

--@client 
-- CODE

--@include path

Mark a file to be included in the upload. This is required to use the file in require() and dofile()

Parameters

  • path ― Path to the file

Usage

--@include lib/someLibrary.txt 
 
require( "lib/someLibrary.txt" ) 
-- CODE

--@includedir path

Mark a directory to be included in the upload. This is optional to include all files in the directory in require() and dofile()

Parameters

  • path ― Path to the directory

Usage

--@includedir lib 
 
require( "lib/someLibraryInLib.txt" ) 
require( "lib/someOtherLibraryInLib.txt" ) 
-- CODE

--@model model

Set the model of the processor entity. This does not set the model of the screen entity

Parameters

  • model ― String of the model

Usage
